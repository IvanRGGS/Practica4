Solucion ejercicios de la Practica 4

Moviendo el trabajo por ahi 1

En este ejercicio se usa el comando git cherry-pick, que sirive para copiar una serie de commits que tu quieras. Por ejemplo, si tienes dos ramas, una main y otra side, y quieres copiar un commit del side al main, tendras que poner el siguiente comando.

git cherry-pick (commit que quieras copiar)

Ejercicio que nos dan

Para este ejercicio nos piden crear una rama adicional con commits de las otras 3 ramas (C3,C4,C7).

Por lo tanto, para resolver el ejercicio habra que usar el siguiente comando.

git cherry-pick C3 C4 C7

Al poner ese comando nos dara el ejercicio como completado y deberia quedar asi.

Simulando la colaboracion

En este ejercicio se usan varios comandos, pero el principal es git fakeTeamwork. Este se encarga de mover el trabajo realizado por otros, o por ti mismo, en un repositorio remoto al principal.

Ejercicio que nos dan

Solucion que nos piden

Para comenzar pirmero tendras que clonar el repositorio que te dan para crear uno aparte, para ello empleamos git clone.

Una vez clonado, tendramos que hacer dos git fakeTeamwork para crear los dos commits externos en el repositorio clonado.

A continuacion haces un commit en el repositorio original para que eso siga siendo en principal commit.

Por ultimo, hacemos un git pull para trar los commits del otro repositorio al principal pero por otra rama diferente del commit que acabamos de hacer.

Como resultado, el ultimo commit de la rama clonada y el ultimo de la rama principal se acabaran juntando en un nuevo commit que sera el commit que contenga toda la informacion.
